## 📈 Key Conclusions & MATLAB Integration
* **Mathematical Modeling:** We translated non-linear differential equations into graphical Simulink blocks to observe real-time ionic gate dynamics 
* **Soma Sensation:** The MATLAB script controls parameters such as injected current ($I_{ext}$), allowing us to analyze cell threshold excitation.
